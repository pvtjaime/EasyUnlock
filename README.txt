Introduction
------------

EasyUnlock is a powerful alternativ to a simple Minecraft
Whitelist. EasyUnlock allows players to explore the world,
but not to build & destroy until they are unlocked.

For installation and usage help, see:
http://www.github.com/hohl/EasyUnlock

Requirements
------------

- Java 6+
- Recent version of Bukkit
